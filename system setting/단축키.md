
#  html 단축키


< html로 작업할때 >

-   html 템플릿 자동 완성 : ! 누르고 엔터
-   중첩 태그 생성 : 태그>태그>태그 ex)

```html
div>li>ol
```

-   인접 태그 생성 : 태그+태그+태그

```html
<div>
<ol>
</ol>
</div>
```

-   아이디설정

```html
div.#name // 클래스 + 아이디 : name
#name // 아이디 : name
```

-   클래스 설정

```html
div.job // 클래스 : job
.job // 클래스 : job
```

-   p태그로 감싸기

```html
p{텍스트 입력} // <p>텍스트 입력</p>
```

** 태그*(숫자) = 태그 반복 가능

-   줄바꿈

```html
태그^^태그2  //태그
              태그
```

-   tab 키 이용 → 태그 끝으로 이동
